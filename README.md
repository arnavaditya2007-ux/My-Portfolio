# Arnav Aditya | Interactive Portfolio 🚀

A premium, interactive cyberpunk-themed portfolio built with modern web technologies. This project showcases the work, skills, and creative journey of **Arnav Aditya**, a Level 18 Developer and CSE student.

## ✨ Features

- **Master Card System**: A persistent, interactive 3D holo-card that follows your journey through the portfolio, reacting to mouse movements and scroll positions.
- **Dynamic 3D Backgrounds**:
  - **Shader Canvas**: A glowing sine wave effect powered by WebGL and Three.js.
  - **Plexus Particles**: An interactive particle web that reacts to the cursor.
- **Interactive Skills Network**: A neural-network-inspired visualization of technical skills using SVG and GSAP, allowing users to explore focus areas like AI, Game Dev, and Web development.
- **Global Reach Globe**: An interactive 3D globe (using `Globe.gl`) showing application deployments and worldwide user reach.
- **Mini-Game**: **Cyber Infiltration**, an integrated hacking mini-game to engage visitors.
- **Smooth Animations**: Powered by **GSAP** and **Lenis** for a high-end, cinematic feel.

## 🛠️ Tech Stack

- **Core**: HTML5, Vanilla CSS3, JavaScript (ES6+)
- **Animation**: [GSAP](https://greensock.com/gsap/) (ScrollTrigger, MotionPath)
- **3D Render**: [Three.js](https://threejs.org/), [Globe.gl](https://globe.gl/)
- **Typography**: Orbitron, Rajdhani (Google Fonts)
- **Icons**: FontAwesome 6
